# Summary of a LIBS Spectrum

Summary of a LIBS Spectrum

## Usage

``` r
# S3 method for class 'libs_spectrum'
summary(object, n_peaks = 5, ...)
```

## Arguments

- object:

  A `libs_spectrum` object.

- n_peaks:

  Integer. Number of top peaks to show. Default 5.

- ...:

  Unused.

## Value

Invisibly returns a list with summary statistics.
