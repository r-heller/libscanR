# Plot a LIBS Spectrum

Plot a LIBS Spectrum

## Usage

``` r
# S3 method for class 'libs_spectrum'
plot(x, y, ...)
```

## Arguments

- x:

  A `libs_spectrum` object.

- y:

  Ignored.

- ...:

  Arguments passed to
  [`ls_plot_spectrum()`](https://r-heller.github.io/libscanR/reference/ls_plot_spectrum.md).

## Value

A `ggplot` object.
