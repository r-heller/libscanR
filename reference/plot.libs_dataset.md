# Plot a LIBS Dataset

Plot a LIBS Dataset

## Usage

``` r
# S3 method for class 'libs_dataset'
plot(x, y, ...)
```

## Arguments

- x:

  A `libs_dataset` object.

- y:

  Ignored.

- ...:

  Passed to
  [`ls_plot_overlay()`](https://r-heller.github.io/libscanR/reference/ls_plot_overlay.md).

## Value

A `ggplot` object.
